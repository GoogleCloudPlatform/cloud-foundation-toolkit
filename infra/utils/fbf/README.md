# Flaky Build Finder

## Usage

```
go install github.com/GoogleCloudPlatformcloud-foundation-toolkit/infra/utils/fbf
```

```
Usage of fbf:
  -end-time string
        Time to stop computing flakes in form MM-DD-YYYY
  -project-id string
        Project ID
  -start-time string
        Time to start computing flakes in form MM-DD-YYYY
  -verbose
        Display detailed table with flaky build IDs
```
